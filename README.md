# 행정구역코드 변환 도구

연도별 행정구역 코드 변경 이력을 조회하고, 두 시점 간 코드 변환을 제공하는 웹 페이지.

**[사이트 바로가기](https://bs-koo.github.io/areacode/)**

## 주요 기능

| 섹션 | 기능 | 설명 |
|------|------|------|
| 변경 이력 | 타임라인 | 2023~2026 주요 행정구역 변경 사항을 시간순으로 확인 |
| 시도코드 조회 | 기준년월 입력 | 특정 시점의 17개 시도코드 조회, 변경 항목 하이라이트, 시도코드/시도명 검색 |
| 지역코드 변환 | 두 시점 비교 | 변경된 코드 매핑표 출력 + Python/JavaScript/SQL 변환 코드 생성 |

## 수록 변경 이벤트

| 시행일 | 이벤트 | 유형 |
|--------|--------|------|
| 2023.06.11 | 강원도 → 강원특별자치도 | 시도코드 42→51 |
| 2023.07.01 | 군위군 경상북도 → 대구광역시 편입 | 시군구 47720→27720 |
| 2024.01.01 | 부천시 원미구·소사구·오정구 재신설 | 시군구 신설 |
| 2024.01.18 | 전라북도 → 전북특별자치도 | 시도코드 45→52 |
| 2026.02.01 | 화성시 만세·효행·병점·동탄구 신설 | 시군구 신설 |

## 기술 스택

- HTML + CSS + JavaScript (순수 바닐라, 프레임워크 없음)
- 데이터: JS 파일 내 JSON 객체로 내장 (DB 불필요)
- 배포: GitHub Pages (정적 파일 서빙)

## 프로젝트 구조

```
areacode/
├── index.html          ← 메인 페이지 (3개 기능 탭)
├── css/style.css       ← 스타일
├── js/
│   ├── data.js         ← 변경 이력 + 시도코드 스냅샷 + 매핑 데이터
│   ├── app.js          ← 메인 로직 (조회, 변환, 검색, 렌더링)
│   └── codegen.js      ← 변환 코드 생성 (Python/JS/SQL)
├── context/            ← 도메인 컨텍스트 문서
└── requirements/       ← 원본 데이터 파일 (xlsx, pdf 등)
```

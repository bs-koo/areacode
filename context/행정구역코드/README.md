# 행정구역코드

연도별 행정구역 코드 변경 이력을 조회하고, 두 시점 간 코드 변환을 제공하는 웹 페이지.

## 배경

한국 행정구역이 연도별로 변경됨에 따라 시도코드, 시군구코드, 법정동코드가 바뀐다.
건축물대장 등 데이터의 앞 5자리(시군구코드)가 달라져서 연도 간 데이터 매칭이 안 되는 문제가 반복적으로 발생한다.
현재는 변경 내역을 매번 수작업으로 찾아보고 있어 비효율적이며, 사업부 다른 개발자들도 동일한 어려움을 겪고 있다.

## 안 하면 어떻게 되는가

- 매번 행정안전부 사이트에서 변경 내역을 수동으로 찾아야 함
- 연도가 다른 데이터 간 매칭 실패 (예: 2022년 데이터와 2025년 데이터의 ADM_DB_PK 불일치)
- 변환 코드를 매번 새로 작성해야 함
- 팀원 간 "이 코드가 뭐였더라" 반복 질문 발생

## 사용자와 규모

- 주 사용자: 데이터랩 팀 (건물 에너지 데이터 분석)
- 부수 사용자: 사업부 내 다른 개발자/연구원
- 대학교 협력 연구원
- 시도코드 17개, 변경 이벤트 5~6건 (2023~2026)

## 성공 기준

- 기준년월 입력 → 해당 시점 시도코드 17개 즉시 확인 가능
- 두 시점 입력 → 변경된 코드 매핑 표 + 변환 코드(Python/JS/SQL) 복사 가능
- 변경 이력을 한눈에 파악 가능

# 행정구역코드 아키텍처

> 전체 구조 요약과 주제별 상세 문서 링크를 관리합니다.

## 기술 스택

- **프론트엔드 전용** (백엔드 없음)
- HTML + CSS + JavaScript (순수 바닐라 또는 경량 프레임워크)
- 데이터: JS 파일 내 JSON 객체로 내장 (DB 불필요)
- 배포: 정적 파일 서빙 (데이터넷 내 페이지 추가 또는 독립 호스팅)

## 시스템 구조

```
areacode/
├── index.html              ← 메인 페이지 (3개 기능 탭 또는 섹션)
├── css/
│   └── style.css           ← 스타일
├── js/
│   ├── data.js             ← 변경 이력 + 시도코드 스냅샷 데이터 (JSON)
│   ├── app.js              ← 메인 로직 (조회, 변환, 렌더링)
│   └── codegen.js          ← 변환 코드 생성 (Python/JS/SQL 문자열)
└── requirements/           ← 요구사항 원본
```

## 화면 구성 (3개 섹션)

### 섹션 1: 행정구역 변경 이력
- 연도별 주요 변경 사항을 타임라인/테이블로 표시
- 변경 전후 코드, 텍스트, 시행일 포함

### 섹션 2: 시도코드 조회
- 기준년월(YYYYMM) 입력
- 해당 시점 유효한 17개 시도코드 + 시도명 테이블 출력
- 변경된 항목 하이라이트

### 섹션 3: 지역코드 변환
- 기준년월1(from), 기준년월2(to) 입력
- 변경된 코드 매핑 표 출력 (시도만 변경이면 시도만, 법정동까지 변경이면 법정동까지)
- 변환 코드 생성 버튼: Python / JavaScript / SQL
- 클립보드 복사 기능

### 주의: 군위군 법정동 리코드 재부여

단순 시도코드 접두사 변경이 아닌, **읍면리 단위에서 리코드가 재부여**된 경우가 있음:
- 예: 소보면 서류리 4772031**036** → 2772031**021** (리코드 036→021)
- 행정동코드도 산성면: 4772**038**000 → 2772**037**000 (읍면동코드 038→037)

# 행정구역코드 아키텍처

> 전체 구조 요약과 주제별 상세 문서 링크를 관리합니다.

## 기술 스택

- **프론트엔드 전용** (백엔드 없음)
- HTML + CSS + JavaScript (순수 바닐라 또는 경량 프레임워크)
- 데이터: JS 파일 내 JSON 객체로 내장 (DB 불필요)
- 배포: 정적 파일 서빙 (데이터넷 내 페이지 추가 또는 독립 호스팅)

## 화면 구성 (4개 섹션)

### 섹션 1: 행정구역 변경 이력
- 연도별 주요 변경 사항을 타임라인/테이블로 표시
- 변경 전후 코드, 텍스트, 시행일 포함

### 섹션 2: 시도코드 조회
- 기준년월(YYYYMM) 입력
- 해당 시점 유효한 17개 시도코드 + 시도명 테이블 출력
- 변경된 항목 하이라이트

### 섹션 3: 지역코드 변환
- 기준년월1(from), 기준년월2(to) 입력
- 변경된 코드 매핑 표 출력 (시도만 변경이면 시도만, 법정동까지 변경이면 법정동까지)
- 변환 코드 생성 버튼: Python / JavaScript / SQL
- 클립보드 복사 기능

### 섹션 4: 지역코드 검색
- 국토교통부 전국 법정동 데이터(약 5만 건)를 클라이언트에서 검색
- 검색어 입력 → 시도명, 시군구명, 읍면동명, 법정동코드에서 LIKE 매칭
- 예: "소사구" 입력 → 시도/시군구/읍면동 컬럼에서 "소사구" 포함 행 출력
- 데이터 소스: `requirements/국토교통부_전국 법정동_20250807.csv` → `js/bjd-data.js`로 변환
- 백엔드 없이 정적 JS 파일로 GitHub Pages 배포 유지

#### 코드 표시 규칙
변경 유형에 따라 의미 있는 자릿수만 표시한다:

| 변경 유형 | 표시 자릿수 | 예시 | 해당 이벤트 |
|----------|-----------|------|-----------|
| 시도 승격 | **2자리** (시도코드) | `42` → `51` | 강원특별자치도, 전북특별자치도 |
| 시군구 편입 | **5자리** (시군구코드) | `47720` → `27720` | 군위군 |
| 구 신설/재신설 | **읍면동 10자리** | `4159025600` → `4159125600` | 화성시, 부천시 |

- 시도 승격: 앞 2자리만 변경, 시군구 전체를 나열하지 않고 시도코드만 표시
- 시군구 편입(군위): 1:1 관계이므로 5자리로 충분. 단, 삼국유사면(`4772038→2772037`, 읍면동 038→037)은 예외로 10자리 포함
- **구 신설/재신설(화성/부천): 1:N 관계(하나의 시군구가 여러 구로 분할)이므로 5자리로는 변환 불가. 읍면동 10자리 필수**
  - 화성: 시군구 재부여(`41590→41591/41593/41595/41597`), 읍면동코드는 유지 (29건)
  - 부천: 시군구+읍면동 **전면 재부여** + 동 이름 변경 (10건 매핑 + 17건 신규)
- 변환 코드 생성(Python/JS/SQL)도 동일 레벨 적용
  - 시도 승격: 2자리 1건
  - 시군구 편입: 5자리 + 뒷자리 변경 예외
  - 구 신설/재신설: 읍면동 10자리 전체 매핑

### 주의: 군위군 법정동 리코드 재부여

단순 시도코드 접두사 변경이 아닌, **읍면리 단위에서 리코드가 재부여**된 경우가 있음:
- 예: 소보면 서류리 4772031**036** → 2772031**021** (리코드 036→021)
- 행정동코드도 산성면: 4772**038**000 → 2772**037**000 (읍면동코드 038→037)

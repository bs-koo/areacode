# 행정구역코드 용어 사전

|------|------|
| 시도코드 | 광역자치단체(시·도) 식별 2자리 코드. 예: 11=서울, 26=부산 |
| 시군구코드 | 시도코드(2) + 시군구(3) = 5자리. 예: 11680=서울 강남구 |
| 법정동코드 | 10자리. 시도(2)+시군구(3)+읍면동(3)+리(2). 법적 행정구역 식별 |
| 행정동코드 | 10자리. 시도(2)+시군구(3)+읍면동(3)+00. 행정 편의상 구역 |
| 기초구역번호 | 우편번호 체계의 5자리 구역 번호 |
| 도로명코드 | 12자리. 시도(2)+시군구(3)+도로명번호(7). 도로명주소 체계 식별 |
| 기준년월 | 코드 조회/변환의 기준이 되는 연월 (YYYYMM 형식) |
